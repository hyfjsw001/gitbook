1.1 shell

Shell

一个bash脚本的正确起始应以#!开头：

#!/bin/bash

在调用bash脚本时候，以#！开头的语句通知系统用何种解释器执行此脚本

# 表示注释

vi 打开，若不存在就创建

$v1  #表明v1是个变量

第一个shell script

#!/bin/bash
#program:
#This program shows "Hello World".
echo -e "Hello World!/n"
exit 0

显示文件权限 

ls -l

更改文件权限

chmod 744 文件名

执行脚本

  sh 脚本名
  ./ 脚本名 需要权限
